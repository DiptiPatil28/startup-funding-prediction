🚀 Startup Funding Prediction
📌 Overview

This project analyzes startup funding data and builds a machine learning model to predict whether a startup is likely to receive high funding or low funding. It demonstrates a complete data science workflow, including data cleaning, exploratory data analysis, model building, evaluation, and basic deployment readiness.

🧩 Problem Statement

The dataset contains information about startup funding rounds, such as industry sector, city location, funding stage, investors, and funding amount.
The objective is to analyze this data to extract meaningful insights and build a machine learning model that classifies startups into high-funding or low-funding categories.

🎯 Key Objectives
Perform data cleaning and preprocessing on real-world startup funding data
Conduct exploratory data analysis (EDA) to understand funding trends
Engineer relevant features for machine learning
Compare multiple machine learning models
Select the best-performing model for prediction
Visualize model performance and prediction confidence
🤖 Machine Learning Models Used
Logistic Regression (baseline model)
Random Forest Classifier (final selected model)

The Random Forest model was selected as the final model because it effectively handles non-linear relationships, performs well on tabular data, is robust to noise, and provides feature importance for interpretability.

📊 Evaluation Metrics
Accuracy
Precision, Recall, and F1-score
Confusion Matrix
ROC Curve and AUC Score
📂 Project Components
Jupyter Notebook containing data analysis, visualizations, and model training
Dataset used for training and evaluation (if permitted)
PowerPoint presentation summarizing the project
Optional Streamlit web application for interactive predictions
⚙️ How to Run the Project
Install the required Python libraries
Open the Jupyter Notebook
Run all cells in sequence
(Optional) Run the Streamlit application for interactive predictions
📈 Conclusion

The Random Forest model outperformed the baseline Logistic Regression model, indicating that ensemble methods are better suited for predicting startup funding outcomes in this dataset. The project provides both analytical insights and a practical predictive solution.

🏆 Project Background

This project was initially developed as part of a codeathon and later refined into a complete GitHub repository for portfolio purposes.
